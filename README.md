# Projeto WIP
Implementação de IA por chat para simplificar o comunicação por whatssap.

# Funcionamento
O projeto em desenvolvimento deverá conter 3 agentes, um irá ser responsavel pela conversa direto com o client em conjunto com outra que será responsavel por análisar a conversa e dizer se a conversa pode progredir ou será nescessário a intervenção de alguem e por ultimo ao final da conversa um modelo irá criar um perfil para tentar indentificar o estilo da pessoa se ela prefere um conversa mais formal ou descontraida.

# Modelo
- Embedding: mxbai-embed-large  
- LLM: llama3.2
